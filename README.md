# Theodore Aware — Portfolio Website

Modern Next.js portfolio built for Theodore Aware.

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development site:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:3000`

## Features

- Next.js 15 App Router
- Tailwind CSS styling
- Component-based architecture
- AI research, expertise, experience, and impact sections
